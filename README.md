# anyRTC-WaWa-Client-Web

## 简介
第三代娃娃机在线解决方案，全新娱乐方式，超低延时娱乐。</br>

## 安装
### 1、编译环境
本地已安装nodejs

### 2、安装依赖
    npm install


## 如何使用？

    npm run dev

    npm run build

### 注册账号
登陆[anyRTC官网](https://www.anyrtc.io/)

### 填写信息
创建应用，在管理中心获取开发者ID，AppID，AppKey，AppToken，替换AppDelegate.h中的相关信息


### 资源中心
[更多详细方法使用，请查看API文档](https://www.anyrtc.io/resoure)
